# **App Name**: DepositEase

## Core Features:

- Input Fields: Accept user inputs for FD amount, interest rate, and FD period using Material UI components.
- Calculation Engine: Calculate the maturity amount and total interest earned based on user inputs.
- Compounding Frequency: Allow users to select the compounding frequency (e.g., quarterly, annually).
- AI Interest Rate Suggestion Tool: Use generative AI to suggest optimal FD interest rates based on current market trends and the FD period the user inputs. Use the selected FD amount, and generate the output considering current market conditions. Note: The user still makes the final decision on rate to use.
- Maturity Projection Display: Display the projected maturity amount and interest earned in a clear and concise format using Material UI components.

## Style Guidelines:

- Background color: Light beige (#FAF9F6) to create a soft, neutral backdrop.
- Primary color: Muted teal (#70A1AF) to evoke trust and stability.
- Accent color: Soft coral (#F08080) for interactive elements like buttons and highlights.
- Body and headline font: 'PT Sans' sans-serif for a clean and modern look.
- Note: currently only Google Fonts are supported.
- Use minimalist icons from Material Design to represent key features.
- Maintain a clean and well-organized layout with sufficient spacing.
